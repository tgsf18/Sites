O que é CSS?

CSS é uma linguagem de estilo usada para definir a aparência de páginas web.

Significado da Sigla

CSS significa Cascading Style Sheets, ou Folhas de Estilo em Cascata.

Formas de Usar CSS

CSS pode ser aplicado de forma inline, interna ou externa.

  • Interno
  • Externo
  • Inline

FUNÇÃO / ANALOGIA

Função

A principal função do CSS é definir o visual e o estilo das páginas da web — cores, fontes, tamanhos, espaçamentos, posições, bordas, animações, etc.

Analogia

HTML é o esqueleto. CSS é a roupa, cabelo e acessórios que dão estilo à estrutura.

Por que separar CSS do HTML?

Organização, reutilização, manutenção e desempenho. Separar o CSS do HTML torna o código mais limpo, rápido e fácil de gerenciar.

O "Chefe" da Web e as 3 Formas de Usar

CSS pode ser usado inline, interno ou externo. Cada forma tem seu uso ideal.

Quem é o W3C?

O W3C (World Wide Web Consortium) define padrões para garantir que os sites funcionem bem, sejam seguros e acessíveis em todos os navegadores.

Por que um programador deve se preocupar em "validar" o código?

  • "Validar" significa verificar se o site segue os padrões do W3C.
  • Evita erros de exibição entre navegadores.
  • Melhora o SEO (posicionamento no Google).
  • Garante acessibilidade e compatibilidade em todos os dispositivos.

EXEMPLOS DE CSS NO HTML

As 3 formas de aplicar CSS

1. Inline

<p style="color:blue;">Texto azul</p>

2. Interno

<style>
  p { color: red; }
</style>

3. Externo

<link rel="stylesheet" href="style.css">

Qual é a mais usada?

A forma externa é a mais comum no mercado por permitir reutilização, organização e manutenção eficiente do código.

A "RECEITA" DO CSS

Anatomia de uma Regra CSS

h1 {
  color: blue;
  font-size: 24px;
}
  • Seletor: h1 — indica quais elementos serão estilizados.
  • Chaves: { } — delimitam o bloco de declarações.
  • Propriedade: color — característica que será alterada.
  • Valor: blue — valor atribuído à propriedade.
  • Declaração: color: blue; — propriedade + valor + ponto e vírgula.

A "MIRA" DO CSS (Seletores Principais)

Seletor De Tag

Seleciona elementos pelo nome da tag HTML Aplica o estilo a TODOS os elementos com essa tag Sintaxe: nome-da-tag { ... }

Seletor de Classe

Seleciona elementos que possuem um atributo class específico Símbolo: Ponto (.) Pode ser aplicado a múltiplos elementos Sintaxe: .nome-da-classe { ... }

Seletor de ID

Seleciona um elemento específico pelo seu atributo id Símbolo: Cerquilha (#) Deve ser ÚNICO na página Sintaxe: #nome-do-id { ... }

HTML correspondente

nav id="menuprincipal">... /nav 
footer id="rodape">... /footer
